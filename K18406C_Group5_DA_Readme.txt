2. Manual and notes
  2.1. Software
	- Visual Studio:
		Windows 7 SP1 (with latest Windows Updates) or higher
		Processor 1.8 GHz or higher
		RAM 2 GB; 8 GB RAM recommended (minimum 2.5 GB if running on a virtual machine)
		Hard disk space: Minimum 800 MB to 210 GB of free space
		.NET Framework 4.5 or higher
		Download (Extension->Manage Extensions): SQL Server Intergartion Services Project
							Microsoft Analysis Services Project	
	-SQL server:
		Os: Windows 7 or later. 
		CPU: Pentium IV or higher.
		Ram: At least 1GB.
		Free disk space: At least 10GB.
		.NET Framework: Version 3.5 SP1 .
		Another tool that comes installed separately is the SQL Server Data Tool.
	 	This tool will help you if you want to work with BI tools like SSAS, SSIS
	-Power BI: 
		For the Desktop version (for desktop and laptop) you will be able to do things like: enter data, connect data models, create and save reports,
 		upload your reports to the site my Power BI web.
		System Requirements:
		Windows 10, Windows Server 2012 R2, Windows Server 2008 R2, Windows Server 2012, Windows 7, Windows 8, Windows 8.1 Power BI for Microsoft Desktop requires Internet Explorer 10 or later.
		Microsoft's Power BI Desktop is available for 32-bit (x86) and 64-bit (x64) platforms.
		.NET: 4.5 RAM: 1GB or more. Monitor: has a minimum resolution of 1440×900 or 1600×900. CPU processing speed: at least 1Ghz.
 		Web browser required: Internet Explorer version 9 or higher or other browsers such as Chrome, Coc Coc, ...
	-Tableau: 
		Operating System: Windows XP/Vista/7/8/8.1/10.
		Memory (RAM): 1 GB RAM required.
		Hard disk space: 450 MB of free space required.
		Processor: Intel Dual Core processor or higher.
	- Excel:
		Install Office 365 Pro Plus hoặc Office 365 E3 to support pivottable
